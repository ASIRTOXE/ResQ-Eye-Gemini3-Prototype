## ðŸš¨ Project Overview
**Eye-ResQ** acts as a **Structural Engineer & Tactical Commander AI**. It transforms passive drone feeds into actionable intelligence for disaster zones (earthquakes, fires).

It analyzes live streams to:
1.  **Calculate Structural Physics:** Predict collapse risks using geometric analysis of cracks and loads.
2.  **Triangulate Survivors:** Fuse visual data with acoustic signals to spot trapped victims.
3.  **Plan Rescue Routes:** Suggest the safest entry points for first responders to avoid secondary collapse.
